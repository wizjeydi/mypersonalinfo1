 updat# Personal Site

> Vite + React + TypeScript scaffolded with Tailwind CSS.

## Quick start

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Files

- [src/App.tsx](src/App.tsx) — main app
- [src/styles/index.css](src/styles/index.css) — Tailwind entry
- [index.html](index.html) — page shell
